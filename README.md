# SpendSphere
SpendSphere is a personal finance and group savings application designed to help users track their budgets and manage group contributions through "Saving Circles".

## Features
- User registration and login.
- Personal budget and expense tracking.
- Saving Circles: Create and join groups to save money together.
- Automated contribution tracking and synchronization with the main dashboard.

## Technologies Used
- Java (Swing for GUI)
- MySQL (Database)
- JDBC (Database Connectivity)
